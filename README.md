# Bienvenue dans notre Tic Tac Toe

Description
- Notre Tic Tac Toe se joue dans une interface graphique
- Il y a deux modes de jeu : 1 joueur contre la machine ou 2 joueurs
- Vous devrez choisir deux chiffres ligne et colonne pour placer votre pion
- Vous pouvez seulement entrer un chiffre entre 1 et 3 tout autre caractère sera refusé

Pré-requis 
- Compiler l'ensemble des fichiers ".gpp"
